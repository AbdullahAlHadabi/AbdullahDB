## Key Concepts Covered

- **INNER JOIN**: Combining tables with matching conditions

- **RIGHT JOIN**: Using right outer joins for complete data analysis
- **Self Joins**: Employee-manager relationships
- **Multiple Joins**: Combining multiple tables
- **String Concatenation**: Building formatted full names
- **NVL() Function**: Handling NULL values
- **CASE Statements**: Conditional logic in queries
- **GROUP BY & Aggregation**: COUNT, SUM operations
- **ORDER BY**: Sorting results by multiple columns
